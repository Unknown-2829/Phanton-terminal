# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 3.x.x   | ✅ Yes             |
| 2.x.x   | ⚠️ Security fixes only |
| < 2.0   | ❌ No              |

## Reporting a Vulnerability

If you discover a security vulnerability:

1. **Do NOT** open a public issue
2. Contact: [@unknownlll2829](https://t.me/unknownlll2829) on Telegram
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact

I will respond within 48 hours and work with you to resolve the issue.

## Security Best Practices

- Only install from official GitHub repository
- Verify the install script before running
- Keep Phantom Terminal updated (`phantom-update`)
